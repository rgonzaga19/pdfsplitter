Privacy Policy

verified_user

Your files never leave your device. All PDF processing runs entirely in your browser — no uploads, no servers, no cloud.

folder_open File Handling

When you open a PDF in PDF Splitter, the file is read directly by your browser using the built-in File API. It is processed in memory using PDF.js and pdf-lib — two open-source libraries that run locally. The file is never transmitted over the internet.

storage Data Storage

  • No cookies are set by this application.
  • No local storage or session storage is used to persist your files.
  • All data is discarded the moment you close or refresh the tab.
  • No analytics, trackers, or third-party scripts collect your behaviour.

public Third-party Resources

This app loads fonts from Google Fonts and icons from Google's Material Symbols CDN. These requests are standard browser fetches and do not include any of your PDF content. If you prefer full offline use, these can be self-hosted.

update Policy Updates

This policy may be updated to reflect changes in the application. The version number in the footer reflects the current release. Last updated: June 2026.
